# Magali Berdah — Plateforme Luxe Mode

Marketplace de vente et enchères de mode et accessoires de luxe.

## 🚀 Démarrage rapide

```bash
# Backend (port 3001)
cd backend && npm install && npx ts-node src/index.ts

# Frontend (port 5173)
cd frontend && npm install && npm run dev
```

## ✨ Fonctionnalités

### Site public
- Home luxe avec enchères en vedette et nouvelles pièces
- Catalogue avec filtres (catégorie, marque, prix, type)
- Page article avec enchères en temps réel
- Inscription / connexion (acheteur ou pro)
- Chatbot mode & luxe
- Newsletter
- CGV internationales (droit français, conformité UE)
- Multilingue FR/EN
- Consentement cookies

### Profil acheteur
- Suivi des achats et enchères
- Favoris
- Notifications

### Espace boutique (Pro)
- Tableau de bord avec statistiques
- Gestion articles (ajout, photos, prix fixe + enchères)
- Gestion commandes + messagerie client
- Portefeuille et revenus
- Abonnement Premium (129€/mois → 5% commission)
- Sans abonnement : 10 articles, 20% commission

### Admin backend
- Dashboard avec toutes les statistiques
- Gestion utilisateurs (vérifier, bannir, supprimer)
- Modération articles (activer, suspendre, mettre en avant)
- Gestion commandes et paiements
- Gestion boutiques et abonnements
- Newsletter (campagnes, abonnés)
- SEO (titre, description, mots-clés, OG)
- Analytics visiteurs et articles
